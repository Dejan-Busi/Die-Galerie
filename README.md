Installationsanleitung
---------------------------
Anforderung: Docker Desktop installiert.

**Schritt 1:** Kopieren Sie den Link des GitHub-Repo's.

<img src="https://github.com/user-attachments/assets/0c2fccee-78a9-45ec-828d-f2cd2a1ae98f" width="500"> <br/><br/>

**Schritt 2:** Klonen Sie das Repo in einem von Ihnen neu erstellten Ordner.

<img src="https://github.com/user-attachments/assets/7dc89b25-edd4-42c0-ba3c-fb35038251f8" width="500"> <br/><br/>

**Schritt 3:** Sämtliche notwendige Dateien und Ordnerstrukturen wurden bei Ihnen heruntergeladen.

<img src="https://github.com/user-attachments/assets/38ce8530-dd0a-4f00-8bf9-a9f6bcce9eb3" width="500"> <br/><br/>

**Schritt 4:** Wechseln Sie das Verzeichnis mit **«cd .\Die-Galerie\»** in den geklonten Ordner.

<img src="https://github.com/user-attachments/assets/01b64942-62a4-477f-b327-1020d6928297" width="500"> <br/><br/>

**Schritt 5:** Geben Sie anschliessend den Befehl **«code .»** ein, um den Ordner in ein neues Fenster zu öffnen. Sie können das alte Fenster schliessen.

<img src="https://github.com/user-attachments/assets/3e3b0ce7-2905-440f-a4ff-5794a6998824" width="500"> <br/><br/>

**Schritt 6:** Vergewissern Sie, dass **Docker-Desktop** am laufen ist.

<img src="https://github.com/user-attachments/assets/39b95a7e-b5c8-454c-aa24-1899776adb80" width="500"> <br/><br/>

**Schritt 7:** Geben Sie den Befehl **«docker-compose build»** ein, damit die Images lokal auf Ihrem Gerät erstellt werden. Die Images sind insgesamt **2.4Gb** gross, der Vorgang kann einige Minuten dauern.

<img src="https://github.com/user-attachments/assets/3327db56-e399-4873-877e-6a079f171afb" width="500"> <br/><br/>

**Schritt 8:** Nachdem die Images erstellt wurden, können Sie die Container mit **«docker-compose up»** starten. Dieser Vorgang kann einige Minuten dauern.

<img src="https://github.com/user-attachments/assets/52becdda-c405-4667-8641-86886fe0a757" width="500"> <br/><br/>

**Letzter Schritt:** Wenn Sie diese Anzeige sehen, wurde alles **erfolgreich** installiert. Sie können die App über die Localhost-Portnummer **4173** öffnen.

<img src="https://github.com/user-attachments/assets/785daee5-4f38-40b4-ba37-f845fcfc7efc" width="500"> <br/><br/>
