Betriebsanleitung:

Schritt 1: Klonen Sie den Code in einem von Ihnen neu erstellten Ordner.

![image](https://github.com/user-attachments/assets/f3bd0c40-3cc9-4e09-a5dd-326413807173)

Schritt 2: Wechseln sie mit "cd" in dem geklonten Ordner.

![image](https://github.com/user-attachments/assets/03e052f6-2d07-4d12-b2a1-228706757491)

Schritt 3: Geben Sie den Befehl «code .» ein, um das Projekt in einem neuen Fenster zu öffnen. Schliessen Sie das alte Fenster.

![image](https://github.com/user-attachments/assets/deabcbd0-bba9-44f6-9328-52e66e88a99e)

Schritt 4: Vergewissern Sie, dass Docker-Desktop am laufen ist.

![image](https://github.com/user-attachments/assets/8b2caca4-b504-46d7-9388-26b7b360fab3)

Schritt 5: Geben Sie den Befehl «docker-compose build» ein, damit die Images lokal auf Ihrem Gerät erstellt werden.

![image](https://github.com/user-attachments/assets/eec7f247-c466-4910-85b4-44d2d6462eb6)

Schritt 6: Nachdem die Images erstellt wurden, können Sie die Container mit «docker-compose up» starten.

![image](https://github.com/user-attachments/assets/254a739b-80f0-41f8-92ae-969b058b2043)

Letzter Schritt: Alle Container der App laufen und Sie können auf die Applikation über die Localhost-Portnummer 4173 zugreifen.

![image](https://github.com/user-attachments/assets/d88e6950-b361-4a8e-8e03-4b1865ea370e)

![image](https://github.com/user-attachments/assets/fd97cb1a-3cc4-4de6-a272-b003522e75c5)


Wichtige Info:

Bitte vergewissern Sie, dass Sie beim Registrieren das Passwort zweimal übereinstimmend eingeben. Wenn das Passwort nicht übereinstimmt, stürzt das Programm aufgrund eines Errors, der nicht mehr rechtzeitig behoben werden konnte, ab.
