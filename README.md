Installationsanleitung
---------------------------
Voraussetzung: Sie haben **Docker Desktop** auf Ihr Gerät installiert   <img src="https://github.com/user-attachments/assets/b9206288-3afc-47f8-b749-3e3f754c6350" width="200"><br/><br/>

**Schritt 1:** Kopieren Sie den Link des GitHub-Repo's.

<img src="https://github.com/user-attachments/assets/05606cda-f243-47d0-9bac-6de8981f3b5d" width="500"> <br/><br/>
 
**Schritt 2:** Klonen Sie das Repo in einem von Ihnen neu erstellten Ordner.

<img src="https://github.com/user-attachments/assets/d0e6c2e1-10dc-4502-a5b4-f848cd73d7da" width="500"> <br/><br/>
 
**Schritt 3:** Sämtliche notwendige Dateien und Ordnerstrukturen wurden bei Ihnen heruntergeladen.

<img src="https://github.com/user-attachments/assets/c031d9ae-a735-4470-9228-069005b51b99" width="500"> <br/><br/>

**Schritt 4:** Wechseln Sie das Verzeichnis mit **«cd .\Die-Galerie\»** in den geklonten Ordner.

<img src="https://github.com/user-attachments/assets/591bf2f0-4eeb-4802-b57c-bfa5131932bc" width="500"> <br/><br/>

**Schritt 5:** Vergewissern Sie, dass Sie **Docker Desktop** gestartet haben.

<img src="https://github.com/user-attachments/assets/b7e39a07-85fc-415b-8df8-fa383cfa9c74" width="500"> <br/><br/>

**Schritt 6:** Geben Sie den Befehl **«docker-compose build»** ein, damit die Images lokal auf Ihr Gerät erstellt werden. Die Images sind insgesamt **2.4Gb** gross, der Vorgang kann einige Minuten dauern.

<img src="https://github.com/user-attachments/assets/9f760c86-a2c9-45d0-8fa6-5eeb43a67333" width="500"> <br/><br/>

**Schritt 7:** Nachdem die Images erstellt wurden, können Sie die Container mit **«docker-compose up»** starten. Dieser Vorgang kann einige Minuten dauern.

<img src="https://github.com/user-attachments/assets/e363160c-ace9-4a00-a962-63ebfa061bcf" width="500"> <br/><br/>

**Letzter Schritt:** Wenn Sie diese Anzeige sehen, wurde alles **erfolgreich** installiert und gestartet. Sie können die App in Ihrem Browser über die Localhost-Portnummer **4173** öffnen.

<img src="https://github.com/user-attachments/assets/760b281e-2879-4ee2-8bde-028943afb0a2" width="500"> <br/><br/>

<img src="https://github.com/user-attachments/assets/0348ed2a-f7b8-49db-b3ae-ef7f64118c0a" width="500"> <br/><br/>
